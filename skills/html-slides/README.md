# HTML Slides

Most slide workflows collapse into either generic templates or brittle screenshots.

This skill builds native single-file HTML decks for executive, advisory, and async review work.

## What It Does

- Classifies the deck mode: template profile, scratch, or web-derived brand.
- Loads the right source context.
- Generates real title-slide previews when style is open.
- Builds a fixed-stage 16:9 HTML deck.
- Runs visual QA for clipping, overflow, contrast, and layout quality.

## Why I Built It

Executive decks are decision artifacts. They need hierarchy, specificity, and polish.

This skill turns slide creation into a repeatable agent workflow without hiding the argument inside exported images.

## Tools It Can Touch

- HTML, CSS, and JavaScript.
- Browser screenshots for visual QA.
- PPTX extraction when a template is provided.
- Web-derived brand discovery when a company URL defines the style.

## Files

- `SKILL.md` - agent instruction.
- `evals.md` - pass/fail quality checks.
- `memory.md` - public development notes.
- `examples/` - sample prompts.
