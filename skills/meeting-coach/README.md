# Meeting Coach

Generic meeting feedback is useless.

This skill reviews transcripts and turns specific moments into better behavior for the next meeting.

## What It Does

- Confirms which speaker is being coached.
- Scores communication behaviors against transcript evidence.
- Identifies moments where the point was buried, the question was soft, or the decision moment was missed.
- Rewrites high-leverage turns in a direct operator voice.
- Gives exactly three concrete reps for the next meeting.

## Why I Built It

The best communication coaching is specific enough to practice.

This skill turns meeting transcripts into a feedback loop for concision, persuasion, question quality, summarization, and decision-driving.

## Tools It Can Touch

- Pasted transcripts.
- Meeting notes or transcript tools when available.
- Voice profile for rewrites.

## Files

- `SKILL.md` - agent instruction.
- `evals.md` - pass/fail quality checks.
- `memory.md` - public development notes.
- `examples/` - sample prompts.
