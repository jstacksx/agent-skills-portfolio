# Question Discipline

Most question lists optimize for coverage. This skill optimizes for leverage.

It helps an agent find the one question that changes the most downstream decisions, then cuts the rest until the ask is sharp enough to use in a real meeting, interview, memo, or discovery process.

## What It Does

- Names the decision the questions are meant to unlock.
- Finds the eigenquestion.
- Ranks supporting questions by decision impact.
- Cuts questions that are interesting but not useful.
- Rewrites the final ask in a direct operator voice.

## Why I Built It

In Corp Dev, advisory, and executive interview work, bad questions waste the room. Good questions expose the constraint, tradeoff, owner, timing, or decision standard.

This skill turns that pattern into a reusable agent workflow.

## Tools It Can Touch

- Workspace context when the question depends on a deal, company, role, or document.
- Connected apps when the relevant context lives in email, calendar, docs, Slack, or meeting notes.
- Voice profile for questions Jann will say or send directly.
